About
SRTHost is a .Net Core application for helping facilitate memory scanning and output for universal speedrunning tools.
Created By: VideoGameRoulette, CursedToast, and Squirrelies
Current Version: 1.0.1.5
How To Install
Install .Net Core Runtime 3.1.X
  • go to the download link provided above for ".Net Core Runtime" and install it to computer it is required to run this application.
    • Install SRTHost
  • go to the download link provided above for "SRTHost Package" and extract it to any folder example lets make a folder called SRTHost on the desktop and drag the all items from the zip file into that folder.
    • Install DirectX Plugin
  • go to the download link provided above for "DirectX Plugin" and take the folder inside the zip and drag it into the plugins folder inside of your SRTHost folder you created.

    • How To Use Web UI
    Web Browser
  • click on the "Web Browser UI" tab above to reveal the SRT UI in your local web browser
    • OBS Studio
  • copy the link provided from "Web Browser UI" tab above and paste it into a web browser source in obs.

    • Custom Params Web UI
    Disables Debug Info
  • debug=1
    • Disables Event Info
  • event=1
    • Disables Stats Info
  • stats=1
    • Disables Position Info
  • position=1
    • Disables Enemy HP Info
  • enemy=1

    • Examples
    Disables All Except PlayerHP
  • ?event=1&position=1&stats=1&debug=1&enemy=1
    • Disables All Except PlayerHP and EnemyHP
  • ?event=1&position=1&stats=1&debug=1
    • Disables Event Info, Position Info, and Debug Info
  • ?event=1&position=1&debug=1
